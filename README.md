# ATM Performance Analysis (SQL + Power BI)

## Project Overview

This project analyzes ATM operational data to understand transaction patterns, revenue generation, operational costs, and profitability. The analysis is structured using **Descriptive Analytics** (what happened) and **Diagnostic Analytics** (why it happened).

SQL was used for data exploration and root-cause analysis, while Power BI was used to build an interactive dashboard for visualization.

---

## Dataset

The dataset contains ATM operational metrics including:

* Financial and Non-Financial Transactions
* Monthly Revenue
* Operational Costs
* Gross Profit
* ATM Uptime
* ATM Type and Location

---

## Tools Used

* MySQL (SQL) – Data exploration and analysis
* Power BI – Dashboard creation and visualization
* Microsoft Excel – Data source

## 🔹 Descriptive Analysis (What Happened?)

Descriptive analysis was performed to summarize historical ATM performance:

* Total transaction volume across ATMs
* Revenue generated by each ATM
* Monthly transaction and revenue trends
* Distribution of financial vs non-financial transactions
* ATM uptime and availability metrics

This helped in understanding overall system performance and identifying high/low performing ATMs.

---

## 🔹 Diagnostic Analysis (Why It Happened?)

Diagnostic analysis was used to identify root causes behind performance variations:

* Comparison of **Revenue vs Operational Cost** to identify profit drivers
* Analysis of low-performing ATMs based on:

  * Low transaction volume
  * Poor uptime
  * Location-based impact
* Correlation between uptime and transaction volume
* Identification of ATMs with high cost but low revenue (inefficiency detection)
* Segmentation of ATMs based on profitability and usage patterns

This helped uncover **why certain ATMs perform better or worse than others**.

---

## Power BI Dashboard

The Power BI dashboard provides interactive insights into:

* ATM transaction trends
* Revenue and cost performance
* Profitability analysis
* Operational efficiency metrics
* ATM performance segmentation

---

## Key Insights

* Identified high revenue-generating ATMs contributing the most to overall profit
* Detected underperforming ATMs with high operational cost and low usage
* Found strong relationship between ATM uptime and transaction volume
* Highlighted location-based performance differences
* Recommended focus areas for improving ATM efficiency and profitability
